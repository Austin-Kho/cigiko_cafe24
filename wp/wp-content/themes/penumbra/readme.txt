== Description == 

Penumbra is a responsive, simple, clean, two columns, customizable and search engine friendly theme. Also supports custom menus, widget ready sidebar and footer, and more customizable feature in the theme options page.

== Installation ==

1. Upload the whole theme folder to your /wp-content/themes/ folder.
2. Activate the theme through the 'Themes' menu in WordPress.

OR

Just grab the Penumbra theme from the built in theme directory and choose "Install now", hit ok, then select the "Activate Theme" link.

== Site Title Limitation ==
* If your site has a very long site title, you may want to disable the top search box if it's cover long site title in some device screen. You can hide or disable top search box in theme option or in theme customizer.
